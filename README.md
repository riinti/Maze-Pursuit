# 3. Key Features
- **키보드를 사용한 조작**:
  - Enter로 게임을 시작합니다.
  - w,a,s,d 또는 방향키를 통해 파란색 공을 움직입니다.

- **점수 집계**:
  - 방문하지 않은 칸을 방문할 때마다 점수가 추가됩니다.

<br/>
<br/>

# 4. Technology
### 4.1 Prim's Algorithm
- **미로 생성 함수**:
  - generate_maze(size)

### 4.2 BFS
- **적과 플레이어 사이 길 탐색 함수**:
  - find_path(start_x, start_y, goal_x, goal_y)
